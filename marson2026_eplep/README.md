# Data source:
https://osf.io/cz9rh/files/osfstorage

# Description
We collected data for almost non-lexical and lexical stimuli across four online studies:
- Exp1 included morphologically simple pseudowords and words
- Exp2 included morphologically complex pseudowords and words
- Exp3 included nonwords and words
- Exp4 included all of the above
In total, the datasets include 1,420,276 data points from 1416 unique participants
