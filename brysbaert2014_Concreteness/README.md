# Reference:
Brysbaert, M., Warriner, A.B., & Kuperman, V. (2014). Concreteness ratings for 40 thousand 
generally known English word lemmas. Behavior Research Methods, 46(3), 904–911. 
https://doi.org/10.3758/s13428-013-0403-5

# Data source:
https://osf.io/qpmf4/

# Description
- Task: Rate how concrete the meaning of each word is on a 5-point scale (1 = abstract, 5 = concrete)
- Participants indicate "N" if they do not know the word well enough to rate it
- 2,008,857 trial-level data points from 4,264 participants (6,069 unique sessions by these ppts)
- 63,039 unique stimulus words
- Data collected via Amazon Mechanical Turk (US-based workers)
- Metadata collected: age, gender, education, handedness, first language, 
  US state raised in, session duration
  
  - NOTE: Participants could do the assignment multiple times. Most did it once, but some did it even more than 
   50 times. This is noted with session_id.
