# Description
##  Experiment 1
- Participants read sequentially presented sentences, where metric stress either aligned or misaligned with syntactic phrase structure, and were asked comprehension probes that tested the effect of this alignment.
- Sentence structures with both subject and object extracted relative clause structures were used as stimuli
- 2920 trials from 40 participants.
- OSF: https://osf.io/42msj
