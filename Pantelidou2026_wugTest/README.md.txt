Description of the study
The multilingual dataset study comprises four datasets (Exp1, Exp2, Exp3, and Exp4), each corresponding to a different language: Catalan, Greek, English, and Spanish, respectively. All datasets follow the same experimental design, differing only in the language of administration. Each experiment consisted of 30 questions. The experiment was implemented using PCIbex Farm, and participants were recruited through Prolific.

Across all languages, the Wug test included an example with a provided answer in each trial to ensure that participants understood the task. This example remained constant throughout the experiment and served solely as a guide; participants could consult it whenever needed. Following the example, participants were presented with a question (the stimulus) and were required to type their response. Each stimulus contained a novel word that participants had to pluralize. After submitting an answer, participants could not return to previous questions, and they were informed of this restriction in advance.

The same test was conducted in six Large Language Models in order to check their linguistic abilities and make comparisons to humans participants. 


For more information, you can visit the osf repository: https://osf.io/4z5n6/overview, which has all relevant details with extra README files.
